# `assistme setup`

Initialize `~/.assistme/assistme.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
assistme setup
assistme setup --workspace ~/.assistme/workspace
```

To run the wizard via setup:

```bash
assistme setup --wizard
```
