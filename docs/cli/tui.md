# `assistme tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
assistme tui
assistme tui --url ws://127.0.0.1:18789 --token <token>
assistme tui --session main --deliver
```
